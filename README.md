## Overview
`calc-vape-wattage` is a simple web application to calculate the output wattage of VAPE devices. By entering the resistance (Î©) and voltage (V), the output wattage is calculated.

## Features
- Calculate output wattage by entering resistance (Î©) and voltage (V)
- Input validation
- Theme switching (light theme and dark theme)
- Save the previous input values to local storage and restore them on the next access
